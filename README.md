# Suede Scan

Free 30-second site scan that funnels into a full graded SEO audit. Suede Labs AI.

Live at [suede-scan.vercel.app](https://suede-scan.vercel.app).

## What it does

Enter a URL, get a fast automated read on technical SEO, metadata, and content signals, then a graded (A–F) audit report — the same grading logic behind Suede's `/suede-visibility-grader` skill, packaged as a public web tool.

## Stack

Node (`server.mjs`), a small `api/` layer, and a static `index.html` front end. No framework build step.

## Run locally

```bash
npm install
npm run dev
```
